# AQI-Prediction-ML-Project
# AQI Prediction using Machine Learning

## Problem Statement
Air pollution is a serious environmental issue affecting human health. Monitoring and predicting Air Quality Index (AQI) helps in taking preventive measures.

## Objective
To build a machine learning model that predicts AQI based on pollutant and weather parameters.

---

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Streamlit

---

## Dataset Features
- PM2.5
- PM10
- NO2
- SO2
- CO
- O3
- Temperature
- Humidity
- WindSpeed
- Date (converted to Month & Day)
